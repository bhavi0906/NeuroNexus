Number Guessing Game
This repository hosts a simple number guessing game implemented in C++. The game generates a random number between 0 and 99, and the player's objective is to guess the correct number. The game provides feedback on each guess, informing the player if their guess is too high or too low.

How to Play
The game generates a random number between 0 and 99.
Enter your guess when prompted.
Based on your guess, the game will inform you if the guess is too high or too low.
Continue guessing until you correctly guess the number.
Features
Utilizes rand() and srand() functions for random number generation.
Provides feedback to the player on each guess, indicating whether the guess is too high or too low.
Clear screen functionality (system("CLS")) for improved user experience.
